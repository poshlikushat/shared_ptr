# shared_ptr
Make a shared pointer
(1) – Мы используем указатель, потому что иначе на стеке будет образоваться каждый раз свой counter, когда мы копируем SharedPtr, оба указателя
должны указывать на один и тот же счетчик. Если бы counter был просто обычной переменной, каждая копия имела бы свой 
отдельный size_t, и счетчик бы не сихнонизировался (каждый из них считал бы себя единсвтенный владельцем)
